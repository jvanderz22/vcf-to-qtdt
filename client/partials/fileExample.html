VCF Input Format

This program takes a file in VCF format and reads the names of all samples named in the file. Any line that begins with ## is a descriptor of the file and is not considered by the program. After these lines, a single line beginning with must be included. This line should contain the following: #CHROM POS ID REF ALT QUAL FILTER INFO FORMAT SAMPLE_X SAMPLE_Y SAMPLE_Z. This line should contain however many samples that are being described by the file. Each subsequent line should contain the information describiing a genetic marker and each sample's genotype at that marker.

Input Sample Data

This program requires the user to input extra data about the samples also. This information can either be entered by hand within the second step of the program or a file can be submitted which will automate the process. If a file is submitted, it should have the same number of lines as there are samples within the VCF file, with each line corresponding to a sample. Each line should contain the following: SAMPLE_ID FAMILY_ID FATHER_ID MOTHER_ID SEX PHENOTYPE. At the end of the file, an empty line is expected to denote that there are no more samples.
