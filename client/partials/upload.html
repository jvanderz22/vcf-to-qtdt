About The Program

This program takes a VCF file and converts it into a set of three files usable by Pedigree Analysis programs such as Merlin. To start, select a VCF file to upload. For an example of a valid, go the example tab here . The next step is to fill the information for each sample ID. This includes the Family ID, the name of the sample's mother and father within the sample, the sex of the person (1 for male or 2 for female), and the person's phenotype for the selected trait (1 for unaffected, 2 for affected, or 0 if the phenotype is unknown). This information may be entered manually on the next page or submitted as a file.After this information is entered and submitted the ouput can be downloaded on the next page.

Program Output

The program outputs files in QTDT format with three seperate output files: a pedigree file, a map file, and a data file. The pedigree file consists of information about each individual and their genotype at each marker. The data file lists all markers in the VCF file and the map file lists the location of each of the markers. This page gives a more complete explanation of these files.
